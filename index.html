What is it?

This section arrives from a completely separate document, this lets us assemble a single page from multiple documents into a single cohesive page.

Using this approach we can color each section differently and effectively circumvent the limitations of Google Docs. We can create a sophisticated document with different people working on each section.

But a big advantage is in the navigation. We can navigate within this document using a menu, just like we can with every normal website.
